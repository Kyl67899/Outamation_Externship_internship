# Extern: Outmation Internship - Data Science
📊 Outmation Internship – Data Science Projects
🔹 Core Pipeline Projects
1. Scanned Document Images Preprocessor
   - Level: Intermediate
   - Tech Stack: Python, Google Colab, PyMuPDF, pdfplumber
   - Description:
       - Build a preprocessing pipeline to clean and prepare scanned document images for downstream tasks like OCR and parsing. Includes noise reduction, format normalization, and PDF extraction.

2. Document Parser
   - Level: Intermediate
   - Tech Stack: Python, OpenCV, PIL, OCR (Tesseract, PaddleOCR, EasyOCR), Google Colab
     Description:
       - Extract structured data from documents using OCR and image preprocessing techniques. Focus on improving accuracy across different document formats.
3. Retrieval-Augmented Generation (RAG) Pipeline
   Level: Intermediate
   - Tech Stack: LlamaIndex, FAISS / Chroma, Python, Google Colab
     - Description:
       - Build a RAG system that retrieves relevant documents and generates context-aware responses using LLMs.
         - Demo: https://youtu.be/WjFtTYO5kzc
4. Retrieval-Augmented Generation (RAG) Pipeline
   - Level: Intermediate
   - Tech Stack: LlamaIndex, FAISS / Chroma, Python, Google Colab
     - Description:
       - Build a RAG system that retrieves relevant documents and generates context-aware responses using LLMs.
       - Demo: https://youtu.be/WjFtTYO5kzc
5. split_and_route() Pipeline
   - Level: Intermediate
   - Tech Stack: Google Gemini API, Mistral, Phi-2, Python
   - Description:
     - Implement a routing system that splits queries and directs them to the most appropriate model or processing pipeline.
6. Search Layer (UI / CLI)
   - Level: Intermediate
   - Tech Stack: Streamlit or Gradio, Python, Google Colab
   - Description:
     Build an interactive interface for querying your pipeline using either a web UI or command-line interface.
7. Final Report & Reflection
   - Description:
     - Document your approach, challenges, learnings, and system design decisions across all projects.
